## 💻 上网方式

### 1. 原生独立IP节点
TikTok 和 Twitter 会严格检测登录 IP。使用廉价的共享机房 IP 极易触发风控导致封号或限流。
* **推荐：** 使用具备“原生住宅”标签的节点，保持一人一号一 IP。
* **线路选择：** [访问 Pika云 获取高匿名专线](https://pikayun.com/)

### 2. 浏览器指纹防护
对于电脑端运营 Twitter 或 TikTok 网页版，建议使用指纹浏览器（如 AdsPower 或 HubStudio）。
* **作用：** 隔离 Cookie 和设备信息，防止多个账号关联被封。

### 3. 移动端环境伪装
* **拔卡：** 手机必须拔除国内 SIM 卡或使用海外流量卡。
* **定位：** 关闭 GPS 定位，并将系统语言、时区修改为节点所在地一致。

## 📖 使用教程

### Clash/V2ray 配置教程
1. 注册并登录 [Pika云](https://pikayun.com/) 获取订阅链接。
2. 打开客户端，点击“配置”或“订阅”，粘贴链接并更新。
3. 开启“全局模式”或“规则模式”进行测试。

### TikTok 零播放/黑屏修复
1. 每次打开应用前，先在浏览器访问 `whoer.net` 检查伪装度是否达到 95% 以上。
2. 若出现黑屏，尝试清除应用缓存，或卸载后更换节点重新安装。

### 推特(X) 安全登录指南
1. 新号登录后前 48 小时请勿大规模改资料或频繁点赞。
2. 建议绑定 Google 邮箱作为恢复项，开启 2FA 二步验证增加安全性。
